**Module 2: `receptra.rag.ingest`** — async ingest_document end-to-end
(RESEARCH §Code Examples — Ingest one document end-to-end, lines 562-621):
- Signature: `async def ingest_document(*, filename: str, content: bytes, embedder: BgeM3Embedder, collection: Collection) -> IngestResult`
- ALLOWED_EXTS = {".md", ".txt"} (D-09 lock); MAX_BYTES = 1_048_576 (1 MiB, D-09 lock)
- Pipeline:
  1. Validate filename ext is in ALLOWED_EXTS → else IngestRejected("unsupported_extension")
  2. Validate len(content) ≤ MAX_BYTES → else IngestRejected("file_too_large")
  3. UTF-8 strict decode → on UnicodeDecodeError raise IngestRejected("encoding_error")
  4. Compute `doc_sha = hashlib.sha256(content).hexdigest()` for stable chunk IDs
  5. `chunks = chunk_hebrew(text)` from Plan 04-02; if empty → IngestRejected("empty_after_chunking")
  6. **Delete-before-add**: query `collection.get(where={"filename": filename})` → if any IDs returned, `collection.delete(ids=existing_ids)` and capture `chunks_replaced` count (RESEARCH §Pitfall 8 mitigation: re-ingest doubles chunk count without delete)
  7. `embeddings = await embedder.embed_batch([c.text for c in chunks])`
  8. `collection.add(ids=[f"{doc_sha[:8]}:{c.chunk_index}" for c in chunks], documents=[c.text for c in chunks], embeddings=embeddings, metadatas=[...])` with metadata schema:
     ```python
     {"filename": filename, "chunk_index": c.chunk_index,
      "char_start": c.char_start, "char_end": c.char_end,
      "doc_sha": doc_sha, "ingested_at_iso": ingested_at,
      "tenant_id": None}  # v2 forward-compat (RESEARCH §Open Decision 4)
     ```
  9. Return `IngestResult(filename, chunks_added=len(chunks), chunks_replaced, bytes_ingested=len(content))`
- Sync ChromaDB calls wrapped via `asyncio.to_thread` (D-03 lock — sync HttpClient + asyncio.to_thread at consumer)

**Module 3: `receptra.rag.retriever`** — async retrieve query → ChunkRef[]
(RESEARCH §Code Examples — Retrieve top-K, lines 624-657):
- Signature: `async def retrieve(*, query: str, top_k: int = 5, embedder: BgeM3Embedder, collection: Collection, min_similarity: float | None = None) -> list[ChunkRef]`
- Returns `list[ChunkRef]` from `receptra.llm.schema` (class identity preserved per Plan 04-01 contract)
- Pipeline:
  1. `qvec = await embedder.embed_one(query)`
  2. `await asyncio.to_thread(collection.query, query_embeddings=[qvec], n_results=top_k, include=["documents", "metadatas", "distances"])`
  3. For each (id, doc, meta, dist) tuple: `similarity = 1.0 - float(dist)`; if `similarity < min_similarity` (default `settings.rag_min_similarity`), skip

  5. Return list (empty list if zero chunks pass threshold — Phase 3 generate_suggestions short-circuits empty list[ChunkRef] to canonical refusal)

    2. **backend/src/receptra/rag/ingest.py** — async ingest_document end-to-end
       (RESEARCH §Code Examples lines 562-621 verbatim, with asyncio.to_thread wrap):

       ```python
       """Hebrew document ingest pipeline (RAG-03 ingest half).

       Pipeline: filename + bytes → ext+size+UTF-8 gates → chunk_hebrew → embed_batch
                → delete-existing-where-filename → collection.add → IngestResult.

       Pitfall #8 mitigation: re-ingest of same filename DELETES prior chunks
       BEFORE adding new ones. chunks_replaced count surfaced to caller.
       """
       from __future__ import annotations

       import asyncio
       import hashlib
       from datetime import UTC, datetime
       from typing import TYPE_CHECKING

       from receptra.rag.chunker import chunk_hebrew
       from receptra.rag.errors import IngestRejected
       from receptra.rag.schema import IngestResult

       if TYPE_CHECKING:
           from chromadb.api.models.Collection import Collection
           from receptra.rag.embeddings import BgeM3Embedder

       ALLOWED_EXTS = frozenset({".md", ".txt"})
       MAX_BYTES = 1_048_576  # 1 MiB (D-09 lock)


       def _validate_extension(filename: str) -> None:
           lower = filename.lower()
           if not any(lower.endswith(ext) for ext in ALLOWED_EXTS):
               raise IngestRejected(
                   code="unsupported_extension",
                   detail=f"Only {sorted(ALLOWED_EXTS)} accepted in v1; got {filename!r}",
               )


       def _validate_size(content: bytes) -> None:
           if len(content) > MAX_BYTES:
               raise IngestRejected(
                   code="file_too_large",
                   detail=f"{len(content)} bytes > {MAX_BYTES} byte limit",
               )


       def _decode_utf8_strict(content: bytes) -> str:
           try:
               return content.decode("utf-8")
           except UnicodeDecodeError as e:
               raise IngestRejected(
                   code="encoding_error",
                   detail="File must be UTF-8 (strict). Save as UTF-8 and re-upload.",
               ) from e


       async def ingest_document(
           *,
           filename: str,
           content: bytes,
           embedder: "BgeM3Embedder",
           collection: "Collection",
       ) -> IngestResult:
           """Ingest one Hebrew .md/.txt document into the RAG collection."""
           _validate_extension(filename)
           _validate_size(content)
           text = _decode_utf8_strict(content)

           doc_sha = hashlib.sha256(content).hexdigest()
           chunks = chunk_hebrew(text)
           if not chunks:
               raise IngestRejected(
                   code="empty_after_chunking",
                   detail="No content after Hebrew normalization + chunking",
               )

           # Delete-before-add: re-ingest of same filename replaces prior chunks
       ```

       ```python
       identity preserved per Plan 04-01 contract — receptra.rag.types.ChunkRef

       """
       from __future__ import annotations

       import asyncio
       from typing import TYPE_CHECKING

       from receptra.config import settings
       from receptra.llm.schema import ChunkRef

       if TYPE_CHECKING:
           from chromadb.api.models.Collection import Collection
           from receptra.rag.embeddings import BgeM3Embedder


       async def retrieve(
           *,
           query: str,
           top_k: int = 5,
           embedder: "BgeM3Embedder",
           collection: "Collection",
           min_similarity: float | None = None,
       ) -> list[ChunkRef]:
           """Return top-K ChunkRefs above the cosine-similarity threshold."""
           threshold = (
               min_similarity if min_similarity is not None else settings.rag_min_similarity
           )

           qvec = await embedder.embed_one(query)
           res = await asyncio.to_thread(
               collection.query,
               query_embeddings=[qvec],
               n_results=top_k,
               include=["documents", "metadatas", "distances"],
           )

           out: list[ChunkRef] = []
           for cid, doc, meta, dist in zip(
               res["ids"][0],
               res["documents"][0],
               res["metadatas"][0],
               res["distances"][0],
               strict=True,
           ):
               similarity = 1.0 - float(dist)  # cosine: distance → similarity
               if similarity < threshold:
                   continue
               out.append(
                   ChunkRef(
                       id=cid,
                       text=doc,
                       source={
                           "filename": str(meta["filename"]),
                           "chunk_index": str(meta["chunk_index"]),
                           "char_start": str(meta["char_start"]),
                           "char_end": str(meta["char_end"]),
                           "similarity": f"{similarity:.3f}",
                       },
                   )
               )
           return out


       __all__ = ["retrieve"]
       ```
