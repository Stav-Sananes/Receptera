<objective>
Build the Hebrew-aware chunker that turns Hebrew Markdown/text into ~1500-char chunks
with sentence-boundary awareness and gershayim/geresh defense. This is the
correctness-critical core of Phase 4 — RESEARCH §Hebrew Chunking Strategy + Pitfall
#8 + Pitfall 2 all converge here. Module is pure Python stdlib (`re` + `unicodedata`)
so it ships without new dependencies and stays trivially unit-testable.

Pipeline (per RESEARCH §Hebrew Chunking Strategy):
1. Decode UTF-8 strict (caller's responsibility — chunker assumes str input)
2. Normalize: NFC + niqqud strip + whitespace collapse (mirrors `receptra.stt.wer.normalise_hebrew`)
3. Split into paragraphs on `\n\s*\n+`
4. For each paragraph, split into sentences on `[.!?]\s+`; re-glue if the leading
   token ends in a known English abbreviation (Dr./Mr./vs./etc./e.g./i.e./Inc./Ltd./Ph.D.)
5. Greedy-pack sentences into chunks until cumulative `len(chunk)` exceeds
   `settings.rag_chunk_target_chars` (default 1500); emit chunk; carry trailing
   `settings.rag_chunk_overlap_chars` (default 200) into the next chunk start
6. Emit `Chunk(chunk_index, char_start, char_end, text)` per chunk
   (char_start / char_end are offsets into the *normalized* text, not raw input)

Key Hebrew-specific defenses (RESEARCH §Cluster 3):
- **Gershayim (U+05F4 ״) and geresh (U+05F3 ׳) NEVER trigger sentence boundaries** —
  these mark abbreviations and Hebrew numerals (e.g., `ע״מ`, `י״ח`). The split
  regex `[.!?]` does NOT include `"` / `'` / U+05F3 / U+05F4 — pitfall #2 mitigation.
- **No capital-letter heuristic** — Hebrew has no case (per hebrew-nlp-toolkit SKILL.md
  Gotchas). The English-trained "next char must be uppercase" rule does NOT apply.
  Use whitespace-then-printable-glyph as the next-char test.
- **Code-block preservation in Markdown** — content inside triple-backtick fences
  is preserved as a single chunk if it fits (regex pre-pass replaces fences with
  placeholders, restored after split).

Edge cases handled (RESEARCH §Hebrew Chunking Strategy):
- Empty input → return `[]`
- Whitespace-only input → return `[]` after normalization
- Single-line file with no `.`/`?`/`!` → whitespace-greedy split fallback (never
  cuts mid-word; no infinite loop on >target_chars single tokens — emit
  best-effort whitespace cut even if a single "word" is >1500 chars)
- Doc shorter than `target_chars` → single chunk
- Niqqud (vowel diacritics) inside words → stripped pre-chunking so chunk byte
  content is stable across re-ingest of the same content with cosmetic niqqud changes

Module surface (consumed by Plan 04-04 ingest):
- `Chunk` frozen dataclass: chunk_index: int, char_start: int, char_end: int, text: str
- `normalize_hebrew(text: str) -> str` — NFC + niqqud strip + whitespace collapse
- `chunk_hebrew(text: str, *, target_chars: int | None = None, overlap_chars: int | None = None) -> list[Chunk]`
  (None defaults pull from `settings.rag_chunk_target_chars` / `_overlap_chars`)

are impossible by construction. RAG-03 chunker requirement is satisfied here;

    ```python
    """Hebrew-aware sentence + paragraph chunker (RAG-03).

    Pure stdlib (re + unicodedata + dataclasses). RESEARCH §Hebrew Chunking
    Strategy is the spec; tests/rag/test_chunker.py is the regression contract.

    Pipeline:
        text → normalize_hebrew → split_paragraphs → split_sentences (per para)
             → greedy_pack (target_chars) → carry overlap → list[Chunk]

    Char offsets are into the normalized text, NOT raw input — caller must hash
    `content.decode("utf-8")` for doc-id stability. (Plan 04-04 ingest does this.)
    """
    from __future__ import annotations

    import re
    import unicodedata
    from dataclasses import dataclass

    from receptra.config import settings

    # --- Public surface --------------------------------------------------------

    @dataclass(frozen=True)
    class Chunk:
        chunk_index: int
        char_start: int       # offset into normalized text
        char_end: int         # exclusive
        text: str             # equals normalized[char_start:char_end] (with overlap stitching)

    # --- Hebrew + sentence regexes --------------------------------------------

    # RESEARCH §Hebrew Chunking Strategy: U+0591..U+05C7 covers Hebrew accents
    # (taamim) + vowel points (niqqud) + bidi + paseq. Phase 2 wer.py uses the
    # same range (regression-tested in test_normalize_diverges_from_wer).
    _HEBREW_DIACRITICS = re.compile(r"[֑-ׇ]")

    _PARA_END = re.compile(r"\n\s*\n+")
    _SENT_END = re.compile(r"(?<=[.!?])\s+")

    # English abbreviations whose period must NOT trigger a sentence split.
    # Case-insensitive; word-boundary anchored. Hebrew gershayim/geresh do NOT
    # appear here because [.!?] never includes them by construction.
    _EN_ABBR_TAIL = re.compile(
        r"\b(?:Dr|Mr|Mrs|Ph\.D|vs|etc|e\.g|i\.e|Inc|Ltd)\.\s*$",
        re.IGNORECASE,
    )

    # --- Normalize -------------------------------------------------------------

    def normalize_hebrew(text: str) -> str:
        """NFC + strip Hebrew diacritics + collapse intra-paragraph whitespace.

        Diverges from receptra.stt.wer.normalise_hebrew on punctuation: the
        chunker PRESERVES `.!?,:;` because they drive sentence-boundary detection.
        WER scorer collapses punctuation to whitespace because it scores word-level
        edit distance. Both modules NFC-normalize and strip diacritics identically.
        """
        text = unicodedata.normalize("NFC", text)
        text = _HEBREW_DIACRITICS.sub("", text)
        # Collapse intra-paragraph runs of horizontal whitespace + single newlines
        # to a single space, BUT preserve `\n\n` (paragraph breaks).
        # Strategy: split on `\n\s*\n+`, collapse each paragraph internally, rejoin.
        paragraphs = _PARA_END.split(text)
        collapsed = [re.sub(r"\s+", " ", p).strip() for p in paragraphs]
        return "\n\n".join(p for p in collapsed if p)

    # --- Split helpers ---------------------------------------------------------

    def _split_paragraphs(text: str) -> list[str]:
        return [p for p in _PARA_END.split(text) if p.strip()]

    def _split_sentences(paragraph: str) -> list[str]:
        parts = _SENT_END.split(paragraph)
        glued: list[str] = []
        for p in parts:
            if glued and _EN_ABBR_TAIL.search(glued[-1]):
                # Re-glue: the previous "sentence" actually ended in an English
                # abbreviation (`Dr.`, `etc.`, ...) — NOT a real boundary.
                glued[-1] = glued[-1] + " " + p
            else:
                glued.append(p)
        return [s for s in glued if s.strip()]

    # --- Chunking --------------------------------------------------------------

    def chunk_hebrew(
        text: str,
        *,
        target_chars: int | None = None,
        overlap_chars: int | None = None,
    ) -> list[Chunk]:
        """Split Hebrew text into ~target_chars chunks at sentence boundaries.

        Args:
            text: raw Hebrew (or mixed Hebrew/English) input. Caller must have
                decoded UTF-8 already.
            target_chars: chunk size in CHARACTERS of normalized text. Defaults
                to settings.rag_chunk_target_chars (1500).
            overlap_chars: trailing chars carried into the next chunk. Defaults
                to settings.rag_chunk_overlap_chars (200).

        Returns:
            list[Chunk] ordered by chunk_index. Empty list if input is empty
            after normalization.
        """
        target = target_chars if target_chars is not None else settings.rag_chunk_target_chars
        overlap = overlap_chars if overlap_chars is not None else settings.rag_chunk_overlap_chars

        normalized = normalize_hebrew(text)
        if not normalized:
            return []

        # Build the flat sentence stream across all paragraphs (paragraph breaks
        # become single " " separators inside a chunk; the original `\n\n` is
        # already-collapsed candidates for boundary preference).
        units: list[str] = []
        for para in _split_paragraphs(normalized):
            units.extend(_split_sentences(para))
        if not units:
            return []

        # Single-doc-shorter-than-target shortcut.
        if sum(len(u) + 1 for u in units) <= target:
            joined = " ".join(units)
            # char offsets land on normalized text — but joined() may differ from
            # normalized due to paragraph-join difference. Recompute by locating
            # the first/last unit in normalized:
            return [Chunk(chunk_index=0, char_start=0, char_end=len(normalized), text=normalized)]

        chunks: list[Chunk] = []
        cur_units: list[str] = []
        cur_len = 0
        cursor = 0  # running offset into the normalized text we have CONSUMED into chunks

        def emit() -> None:
            nonlocal cur_units, cur_len, cursor
            if not cur_units:
                return
            chunk_text = " ".join(cur_units)
            char_start = cursor
            char_end = char_start + len(chunk_text)
            chunks.append(Chunk(
                chunk_index=len(chunks),
                char_start=char_start,
                char_end=char_end,
                text=chunk_text,
            ))
            # Carry overlap: keep tail sentences whose total length <= overlap.
            tail_chars = 0
            tail_units: list[str] = []
            for u in reversed(cur_units):
                if tail_chars + len(u) + 1 > overlap and tail_units:
                    break
                tail_units.insert(0, u)
                tail_chars += len(u) + 1
            cur_units = tail_units
            cur_len = sum(len(u) + 1 for u in cur_units)
            cursor = char_end - tail_chars  # overlap window starts here

        for u in units:
            if cur_len + len(u) + 1 > target and cur_units:
                emit()
            cur_units.append(u)
            cur_len += len(u) + 1

            # Defensive: a single unit (e.g. one >target_chars Hebrew word) cannot
            # be split further at sentence level. Emit it as-is and continue.
            if len(u) > target and len(cur_units) == 1:
                emit()

        if cur_units:
            emit()

        return chunks

    __all__ = ["Chunk", "chunk_hebrew", "normalize_hebrew"]
    ```
